Enhance the project detail modals by adding a visual workflow / timeline section for every project.

Inside each project modal, add a new section called:

Workflow Overview

This section should visually represent the project process as a horizontal workflow diagram.

Each step should appear as a card connected with arrows.

Use the following structure:

Step → Step → Step → Step

The steps should have:
• Icon
• Title
• Short description

Ensure the design matches the dark theme and looks modern.
